## 🎯 Objetivo

Visualizar e gerenciar todos os detalhes de um projeto específico, suas sprints, cards e membros.

---

## 📄 Componentes da Tela

### Header / Informações gerais

- Logo
- Nome do projeto
- Status (ativo / inativo / arquivado)
- Criador do projeto
- Datas de criação e atualização
- Botões de ação: Editar, Arquivar, Excluir (conforme permissões)
### [[Dashboard - Projeto]]

## Sprints do Projeto (`projeto/sprints.md`)

- Lista de sprints do projeto (cards ou tabela)
- Informações principais: nome, status, período
- Botão para abrir sprint (redireciona para o board da sprint que tem um Kanban)
- Filtros: ativa, concluída, futura; por usuário; por data
- [[Dashboard - Sprint]]
- Dentro dos cards:
  - [[Card]]
  - [[Upload - Download de Arquivos]]

## Configurações do Projeto (`projeto/configuracoes.md`)

- Informações do projeto: nome, logo, descrição, status, datas, criador
- Botões de ação: editar, arquivar, excluir (conforme permissões)
- Painel de membros:
    - Lista de usuários com papéis
    - Adicionar/remover membros
    - Filtro por papel ou status de usuário
- Gestão de permissões por sprint ou global
- Upload/download de anexos do projeto (restrições e segurança)
- Logs e histórico de alterações do projeto
