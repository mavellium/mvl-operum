

### 🎯 Objetivo

Exibir notificações do sistema, alertas críticos e feedbacks de usuários.

### 📄 Componentes

- **Lista de Notificações**
    - Tipo: alerta, informação, sucesso, erro
    - Origem: projeto, sprint, usuário
    - Status: lida / não lida
- **Detalhes**
    - Conteúdo completo da notificação
    - Link para ação relacionada (ex.: abrir sprint, card ou projeto)
- **Configurações**
    - Marcar todas como lidas
    - Filtrar por tipo, projeto ou usuário
    - Preferências de notificação por email ou push
